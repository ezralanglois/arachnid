enhance_volume

Volume filtering and amplitude enhancement

This pySPIDER batch file (spi-enhance) filters (and enhances) a volume to a specific resolution.

Tips

  1. Boolean flags such as disable-enhance cannot take a value on the command-line. They only require a value in a configuration file. Their existence or absence sets the appropriate value. For example, specifiying $ spi-filtervol –disable-enhance .... will disable amplitude enhancement.
  2. This script interpolates passed the last value in the scatter file. It uses a linear model that starts at 14 A and goes to the last available resolution.

Examples

# Source AutoPart - FrankLab only

$ source /guam.raid.cluster.software/arachnid/arachnid.rc

# Filter and enhance several volumes

$ spi-enhance vol1.spi vol2.spi -p params.spi -o filt_vol_0001.spi -r 15 --scatter-doc scattering8.spi

Critical Options

-i <FILENAME1,FILENAME2>, --input-files <FILENAME1,FILENAME2>, FILENAME1 FILENAME2

List of input filenames containing volumes. If you use the parameters -i or –inputfiles they must be comma separated (no spaces). If you do not use a flag, then separate by spaces. For a very large number of files (>5000) use -i “filename*”

-o <FILENAME>, --output <FILENAME>

Output filename for filtered/enhanced volume with correct number of digits (e.g. masked_0000.spi)

-p <FILENAME>, --param-file <FILENAME>

Path to SPIDER params file

--bin-factor <FLOAT>

Number of times to decimate params file

--resolution <FLOAT>

Resolution to filter the volumes

--scatter-doc <FILENAME>

Filename for x-ray scatter file

Tight Masking Options

--enh-mask <BOOL>

Generate a tight mask under which to calculate the fall off

--enh-filter <FLOAT>

Gaussian lowpass filter to given resolution for tight mask (0 disables)

--enh-gk-sigma <FLOAT>

Gaussian filter width in real space for tight mask (0 disables)

--enh-gk-size <INT>

Size of the real space Gaussian kernel for tight mask (must be odd!)

--enh-threshold <'A' or FLOAT>

Threshold for density or A for auto threshold for tight mask

--enh-ndilate <INT>

Number of times to dilate the mask for tight mask
