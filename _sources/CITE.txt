How to cite
===========

The main reference to cite is:


	Langlois, R., Ash, Ho N., Frank, J., 2014. Arachnid: Automated 
	Image-processing for Electron Microscopy. In Preparation.

You can find this reference along with other references to specific
programs :ref:`below <citation-manager>` in a file format that is easy to import into your
favorite citation manager.
